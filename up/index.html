Case Study: Union Pacific Railroads

Bilal M. Ayyub (PI); Yating Zhang; Sherief Elsibaie; Yujie Mao; Majed Hamed; Magdy Elsibaie

Union Pacific (UP) is the largest railroad in North America, covering 23 states in the western two-thirds of the United States.

Terminology

  • A Network is any connected system, where connection points can be repesented by nodes and connections can be represented by links.
  • A Path between node i and node j is the number of links in a set of links that connect the two nodes. The Shortest Path between node i and node j is the smallest path of all paths between node i and node j.
  • Node eccentricity of node i is the maximum value of the set of shortest paths (in link units) from node i to all other nodes. Eccentricity can be used to compute the network's radius, which is its minimum node eccentricity and diameter, which is the maximum node eccentricity, of a network as well as identify its geometric center.
  • Pairwise efficiency is a charactertistic between any node i to node j and characterizes how well connected they are to each other. It is calculated as the reciprocal of the shortest path (in link units) between those two nodes.
  • Node efficiency of a node i characterizes how well connected it is to all other nodes in a network. It is calculated as the average of all node pairwise efficiency values between node i and all other nodes.
  • Network efficiency of a network characterizes how well connected the network is in general. It is calculated as the average of all node efficiency values for all nodes on the network.
