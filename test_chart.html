Rail Network Topological Analysis and Throughput Assessment (RailNTA)

Team members: Bilal M. Ayyub; Yujie Mao; Sherief Elsibaie; Majed Hamed; Magdy Elsibaie; Yating Zhang

Objectives

  • Implement and use a network analysis Python computer code, designed to quantifying the topological connectedness efficiency of freight railroad networks for resilience studies
  • Design the architecture of a Tool for Rail Network Topological Analysis and Throughput Assessment (RailNTA)
  • Use aggregate network to illustrate the methodology and the tools, and offer a proof of concept in coordination with FRA and industry partners

Requirements

  • Assess topological connectedness efficiency necessary for resilience studies
  • Nodal and link criticality and failure impacts
  • Develop environment, input and output
  • Modular components
  • Graphical user interface (GUI)
  • Environment for hosting (free)
  • Develop database: input and output (TBD)
  • Develop GIS capabilities (TBD)
  • Establish proof of concept
  • Illustrate (example) and demonstrate (case studies)
